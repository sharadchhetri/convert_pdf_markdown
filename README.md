# convert_pdf_markdown
This python script will help to convert from PDF to markdown file.

## Python Virtual Environment

Run the command for creating Python Virtual Environment

```
python -m venv .venv
```

Activate the Virtual Environment (For Linux\Unix and Mac OS)

```
source .venv\bin\activate.sh
```

## Install the python libraries

Install python libraries.
```
pip install -r requirements.txt
```

## Execute convert_pdf_markdown.py

1. Upload the pdf file in your system.

2. For executing the python script, run this command
```
python convert_pdf_markdown.py
```

**Note:** A sample pdf file is also included in the repo.
